# 外部動画生成API比較 (2025年12月更新)

## 要件
- **8秒の縦型動画 (9:16)**
- **テキストプロンプトから動画生成**
- **音楽付き**
- **月120本 (毎日4本)**

---

## 候補1: **Fal.ai - Google Veo 3** (推奨🌟)

### 特徴
- Gemini Veo 3.1と同等のモデル (Google製)
- 音声付きで高品質
- API統合が簡単

### 価格
- **Veo 3 Standard**: $0.75/秒 (音声あり)
- **Veo 3 Fast**: $0.40/秒 (音声あり) ← **推奨**

### 月間コスト (120本 x 8秒)
- **Veo 3 Fast**: $0.40/秒 × 8秒 × 120本 = **$384/月**
- **Veo 3 Standard**: $0.75/秒 × 8秒 × 120本 = **$720/月**

**API URL**: https://fal.ai/models/fal-ai/veo3

---

## 候補2: **Replicate - Kling v1.6**

### 特徴
- 高品質な動画生成
- 価格が比較的安価
- 音楽は別途生成が必要

### 価格
- **Kling v1.6 Standard**: $0.28/5秒 = $0.056/秒
- **Kling v1.6 Pro**: $0.098/秒

### 月間コスト (120本 x 8秒)
- **Kling Standard**: $0.056/秒 × 8秒 × 120本 = **$53.76/月**
- **音楽生成 (別途)**: 約$10-20/月
- **合計**: **約$70/月**

**API URL**: https://replicate.com/collections/text-to-video

## 候補3: **Fal.ai - Kling AI**

### 特徴
- Replicateより安価
- 5秒単位の課金
- 音楽は別途生成が必要

### 価格
- **$0.14/unit** (1 unit = 5秒の動画)
- 8秒の動画 = 2 units = $0.28/本

### 月間コスト (120本)
- $0.28/本 × 120本 = **$33.60/月**
- **音楽生成 (別途)**: 約$10-20/月
- **合計**: **約$50/月**

**API URL**: https://fal.ai/models (Kling AI)
